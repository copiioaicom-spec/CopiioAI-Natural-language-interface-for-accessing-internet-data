# VitePress SEO Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the existing Docsify setup with a VitePress static site that gives each docs page an independent, Google-indexable URL, deployed automatically via GitHub Actions to GitHub Pages.

**Architecture:** VitePress builds `docs/` into static HTML in `docs/.vitepress/dist/`. GitHub Actions pushes that dist folder to the `gh-pages` branch on every push to `main`. GitHub Pages serves from `gh-pages`. The existing 7 Markdown files are kept as-is with frontmatter added for per-page SEO meta tags.

**Tech Stack:** VitePress 1.6.x, Node.js 20, GitHub Actions (`peaceiris/actions-gh-pages@v3`), GitHub Pages.

**Note on repo name:** Current remote is `copiioaicom-spec/copiioai`. If you renamed the repo on GitHub to `natural-language-interface-for-internet-data`, replace every occurrence of `/copiioai/` in this plan with `/natural-language-interface-for-internet-data/`.

## Task 7: Create GitHub Actions Deploy Workflow

**Files:**
- Create: `.github/workflows/deploy.yml`

- [ ] **Step 1: Create the workflows directory**

```bash
mkdir -p /Users/cid/Desktop/workspace/copiioai/.github/workflows
```

- [ ] **Step 2: Write .github/workflows/deploy.yml**

Write the following to `/Users/cid/Desktop/workspace/copiioai/.github/workflows/deploy.yml`:

```yaml
name: Deploy VitePress to GitHub Pages

on:
  push:
    branches: [main]
  workflow_dispatch:

permissions:
  contents: write

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm

      - run: npm ci

      - run: npm run docs:build

      - uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: docs/.vitepress/dist
```

- [ ] **Step 3: Commit and push**

```bash
cd /Users/cid/Desktop/workspace/copiioai
git add .github/workflows/deploy.yml
git commit -m "ci: add GitHub Actions workflow to build and deploy VitePress to gh-pages"
git push origin main
```
